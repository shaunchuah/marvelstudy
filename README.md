# MARVEL Study Website

Website for MARVEL IBD study.

## Framework

- NextJS
- TailwindCSS

## Development Setup

Install nodejs.

```sh
npm install
npm run dev
```

## Deployment Plan

Vercel

## Project Structure

- `design/` All design assets
- `pages/` NextJS routes
- `public/` All static assets
- `styles/` Globals.css to initialize TailwindCSS
